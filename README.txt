== Description ==

## Plugin Savage Note pour wordpress

Le Plugin Savage Note pour WordPress vous permet de connecter votre site web à l'API Savage Note, vous offrant ainsi un accès à une vaste bibliothèque d'articles optimisés pour le SEO que vous pouvez acheter et importer directement dans votre site WordPress.

## Fonctionnalités

- Configuration facile : il vous suffit de saisir votre clé API pour commencer à parcourir les articles de la bibliothèque de Savage Note.
- Recherche et filtre des articles par mot-clé, catégorie, thématique et plus encore.
- Achat directement depuis l'interface du plugin et importation des articles dans votre site en un clic.
- Intégration avec les paramètres de référencement de votre site WordPress pour assurer une visibilité optimale dans les moteurs de recherche.

## Pré-Requis

Pour utiliser le plugin, vous aurez besoin de :
- Une clé API valide de Savage Note.
- Un site WordPress fonctionnant avec la version 6.0 ou ultérieure.

== Installation ==

1. Téléchargez la dernière version du plugin à partir du dépôt de plugins WordPress ([ici](https://github.com/QuentinMkr/sn-plugin)).
2. Téléversez les fichiers du plugin dans le répertoire /wp-content/plugins/sn-plugin, ou installez le plugin directement à partir de l'écran des plugins de WordPress.
3. Activez le plugin depuis l'écran "Plugins" de WordPress.
4. Saisissez votre clé API sur la page des paramètres du plugin.
5. Commencez à parcourir, importer et à acheter des articles !

== Frequently Asked Questions ==

= Comment puis-je obtenir une clé API de Savage Note ? =

Pour obtenir une clé API de Savage Note, vous devez vous inscrire sur le site web de [Savage Note](https://savage-note.com) et suivre les instructions pour créer un compte.

= Comment puis-je trouver des articles pertinents pour mon site web ? =

Vous pouvez utiliser la fonction de recherche de la partie "Acheter des articles" pour trouver des articles en fonction de mots-clés, de catégories ou de thèmes pertinents pour votre site web. Vous pouvez également prévisualiser les articles avant de les acheter pour vous assurer qu'ils répondent à vos besoins sur [notre site](https://savage-note.com).

= Comment puis-je intégrer les articles achetés à mon site web ? =

Une fois que vous avez acheté des articles, vous pouvez les importer directement dans votre site WordPress en un clic. L'article sera alors automatiquement intégré à votre bibliothèque d'articles et pourra être modifié, publié ou programmé pour une publication future selon vos besoins. Les images seront également sauvegardées sur votre site.
